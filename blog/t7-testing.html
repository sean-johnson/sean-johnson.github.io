What is scope?

I translate scope to mean the amount of access one has to variables, objects and functions within a context. In Javascript, It relates to the ability for variables to have 'local' or 'global' accessibility (scope) when referenced or called and in what context it can be executed. For example, if a variable is defined within a function's curly braces {}, the variable's accessibility is confined 'locally' within that block of code. The scope is kind of like a hierarchical order that determines a variable's accessibility. The best way to explain is with these examples taken from www.w3schools.com.

Global scope:

var carName = " Volvo";
// code here can use carName

function myFunction() {
// code here can use carName
}

Local scope:

// code here can NOT use carName

function myFunction() {
var carName = "Volvo";
// code here can use carName
}
