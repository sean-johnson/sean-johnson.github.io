Parentheses ( )

Parentheses are used in functions, to define a function's parameters. Example:

myFunction(parameterOne, parameterTwoEtc) {code};

They can be used to group together a expression to help the computer calculate a value from a statement:

2 * (4 + 7) =

They can be used to to define a situation in a conditional statement. (if / else, for, while etc):

if (conditional parameter goes here) {do this instruction};

Brackets [ ]

Brackets are used in arrays. They can store values in an indexed format (starting at zero for the first value in the array):

var myArray = [1, "two", 3, 4.0, "and", "so", "on"];

They can be used to extract these values by referencing their index number:

myArray[1] === "two"

Braces { }

Braces are usually used to block out code into objects or 'blocks' of code. It helps in terms of formatting and structure. It can be used to identify a function's instructions:

myFunction() {instructions goes here};

It can be used to wrap statements (or instructions) in a if / else conditional statement:

if (a === 1) {code goes here};

They basically encapsulate code into objects or blocks of code that help to create clean, readable and structured code. They also help to create 'enviroments' where variables can be created locally (within the braces) and that code will only be referenced (or usable) from within that object.

'Single' and "double" quotes

Interchangeably, these can be used to identify strings or string literal statements. A string is basically a word or words that are interpeted as a value of STRING by the computer. There are different types of values to be interpreted —strings are one example. An number (1) is another type of value and is a different type than a string. A string needs to be identified with single or double quotes to be defined as a string. The number "1" can be a string value or the number 1 can be a number value; it depends on how it is defined. For example:

var myString = "Here is a string"

var myOtherString = '1'

var myNumber = 1
