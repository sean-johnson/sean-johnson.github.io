.map(), .filter(), .reduce()

.map() I think creates a new array of values by calling (or executing) a chosen function for each item in a already defined array?
.filter() finds an element/s (or item/s) within an array of a specific description.
.reduce() goes and combines defined values in a array from left to right and outputs the end result.
